
## 1. 개요

###  1) kafka 소개
   
    Apache Kafka (아파치 카프카)는 분산형 스트리밍 플랫폼입니다. Linkedin에서 개발되었고 2011년에 오픈소스로 공개되었습니다.
    
###  2) kafka 특징
   
    하나 이상의 클러스터 구성
    Topic 에 레코드 스트림을 저장
    Partition으로 분할 ( topic을 멀티 파티션으로 생성할 경우 순서를 
    Replication으로 복제 
    
    레코드 = (key , value , timestamp)
